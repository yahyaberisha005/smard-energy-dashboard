# SMARD Energy Dashboard (Germany)

Portfolio project: A Power BI dashboard analyzing Germany’s electricity market using SMARD (Bundesnetzagentur) data.
Focus: Day-ahead price, generation mix (wind/solar), demand, and key drivers over time.

## Data source
- SMARD electricity market data (Bundesnetzagentur): https://www.smard.de/en/

## Key questions
- How do day-ahead prices evolve over time (MoM/YoY)?
- How do wind/solar generation and demand correlate with price spikes?
- Which periods show extreme volatility and what are the likely drivers?

## Deliverables
- Power BI report (.pbix) with an Overview page + Driver pages
- SQL query pack used for transformations / KPI calculations
- 1-page case study (problem → approach → insights → recommendations)
